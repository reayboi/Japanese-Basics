It's time to revise really basic words and phrase that you should know by now.

こんにちは = hello (generalised)
おはよ = good morning
こんんぼわ = good afternoon
ありがとう = thank you
またね = see you later
さようなら = goodbye (for a long time)
おやすみ = good night
ごめんなさい = I'm sorry
どうも = okay/thanks (an acknowledgement)

なんですか = what is it?
これはペンです = This is a pen
お元気ですか = how are you?
私はとても元気です = I'm very well
